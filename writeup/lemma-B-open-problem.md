# Lemma B: proving `H¹(X, F₂)` is indecomposable for 2-group Belyi maps

This is the sole remaining gap in the negative resolution of Musty Q1.2.5 (see
`writeup/main-result.md`). It is a problem in the **modular representation theory of finite
2-groups** — specifically about a distinguished quotient of the Gruenberg relation module.
Overwhelming computational evidence (2008/2008 cases, `|G| ≤ 128`); no proof yet.

---

## 1. Precise statement

Let `F = F₂ = ⟨x, y⟩` be free of rank 2, `π: F ↠ G` a surjection onto a finite 2-group with
kernel `N = ker π`, and let `φ: X → P¹` be the corresponding Galois cover branched over
`{0, 1, ∞}` with deck group `G` and inertia generators `σ₀ = π(x)`, `σ₁ = π(y)`,
`σ∞ = π((xy)⁻¹)`.

> **Lemma B.** `H¹(X, F₂)`, with its natural `G`-action, is an **indecomposable**
> `F₂[G]`-module. Equivalently, `End_{F₂[G]}(H¹(X, F₂))` has no idempotents other than
> `0` and `1` (then Lemma A, already Lean-verified, finishes the arithmetic conclusion).

Because the only simple `F₂[G]`-module is the trivial one (G is a 2-group), all the
subtlety is in how the composition factors are glued.

---

## 2. The module-theoretic setup (Gruenberg relation module)

**Open curve.** Let `X° = X ∖ φ⁻¹{0,1,∞}`. Then `π₁(X°) = N`, and by Nielsen–Schreier
`N` is free of rank

    rank(N) = 1 + |G|·(2 − 1) = 1 + |G|.

Hence `H¹(X°, F₂) = N^{ab} ⊗ F₂` has dimension `1 + |G|` over `F₂`. As an `F₂[G]`-module this
is the **relation module**, and it sits in the reduction mod 2 of the **Gruenberg
resolution** of `G` from the presentation `F ↠ G` (`F` free of rank 2):

    0 → N^{ab} → Z[G]² → Z[G] → Z → 0.

Splitting off the last map (`Z[G] → Z` is augmentation with kernel `I_G`) gives, after
`⊗F₂`,

    0 → H¹(X°, F₂) → F₂[G]² → I_G → 0,          (†)

where `I_G = ker(F₂[G] → F₂)` is the augmentation ideal. So `H¹(X°, F₂)` is the kernel of a
surjection from the **free** module `F₂[G]²` onto `I_G`.

**Closed curve (filling punctures).** The map `H¹(X°, F₂) ↠ H¹(X, F₂)` is surjective with
kernel spanned by the classes of loops around the punctures, modulo the single relation
"sum of all puncture loops = 0". The punctures over branch point `b` form one `G`-orbit
`G/⟨σ_b⟩` (`e_b = ord σ_b`), so:

    dim H¹(X, F₂) = 2g,   2g = 2 + |G|·(1 − 1/e₀ − 1/e₁ − 1/e∞)   (Riemann–Hurwitz),
    dim ker[H¹(X°) ↠ H¹(X)] = (Σ_b |G|/e_b) − 1.

The kernel is the `F₂[G]`-submodule generated by the punctures: `⊕_b Ind_{⟨σ_b⟩}^G F₂`
modulo one trivial quotient. So

> **`H¹(X, F₂)` is the quotient of `ker(F₂[G]² ↠ I_G)` by the (image of the) puncture
> module `(⊕_b F₂[G/⟨σ_b⟩]) / F₂`.**

This is a completely explicit description of the module whose indecomposability we need.
Everything is finite-dimensional linear algebra over `F₂[G]`; the challenge is structural,
not computational.

---

## 3. What has been ruled out

- **Simple socle is FALSE.** A 2-group module has simple socle iff `dim M^G = 1`, which would
  give indecomposability for free. But `dim H¹(X, F₂)^G` reaches **3** (e.g. `[32,5]` quo 1,
  genus 9). So the easy proof does not apply.
- **Simple top is also FALSE.** `H¹(X, F₂)` is **self-dual** (the Weil/cup-product pairing on
  `J[2]` is `G`-invariant and nondegenerate), so `dim M_G = dim M^G`; a socle of dim 3 forces
  a top of dim 3 too.

So the indecomposability is genuine but *not* detected by either the socle or the top being
simple. Any proof must use the specific gluing in (†) + the puncture quotient, not a soft
socle/top argument.

---

## 4. Suggested attack, easiest first

**(a) Cyclic `G = C_{2^k}` (do this first).** Here `F₂[G] = F₂[t]/(t^{2^k})` (`t = σ − 1`)
is a local **uniserial** ring, and *every* finitely generated module is a direct sum of the
uniserial pieces `F₂[t]/(t^j)`, `1 ≤ j ≤ 2^k`. So `H¹(X, F₂)` is indecomposable **iff it is
a single uniserial piece iff it is a cyclic `F₂[G]`-module** (generated by one element). For
cyclic `G` the curves are the explicit superelliptic `y^{2^k} = c·xᵃ(x−1)ᵇ` (see
`belyi/cyclic.sage`); computing the module directly and checking it is `F₂[G]`-cyclic should
be a short, self-contained proof — and a genuine first theorem.

**(b) Metacyclic / split `C_{2^a} ⋊ C_{2^b}`.** These dominate the census and always came
out indecomposable. Use the exact sequence (†): `F₂[G]²` is free, `I_G` is the augmentation
ideal; understand the puncture quotient explicitly and show no idempotent of the quotient
lifts to a splitting. The `Ext¹`/gluing data here is tractable for metacyclic 2-groups.

**(c) General 2-group.** The representation type of `F₂[G]` is *wild* for most 2-groups, so a
classification of all modules is hopeless — but indecomposability of **this specific**
module may still follow from its construction. Candidate structural reasons to test:
  - **Connectedness.** `X` is connected (`G` is 2-generated, monodromy transitive), i.e.
    `H⁰(X,F₂) = F₂`. Does connectedness of the cover force `H¹` indecomposable? (Speculative;
    worth probing on the data — is there a decomposable `H¹` for a *disconnected* or
    non-2-generated setup, to see whether 2-generation is the operative hypothesis?)
  - **The relation module is "as free as possible."** By Gaschütz/Gruenberg theory the
    relation module `N^{ab} ⊗ F₂` of a `d`-generated group is stably free-ish; the failure to
    decompose after the puncture quotient may be pinned by a single non-split extension class
    in `Ext¹_{F₂[G]}(I_G, puncture module)`.

**Reduction target for Lean:** once (a)/(b) yield a uniform reason, the statement "the
quotient module in §2 has only trivial idempotents" is itself formalizable, and would plug
directly into the already-proved `isLocalRing_end_of_indecomposable` bridge.

## 5. Literature to engage

Wild/modular `G`-cover cohomology structure (the closest existing theory):
- **K. Gruenberg**, *Relation modules of finite groups* (CBMS 25, 1976) — the (†) framework.
- **W. Gaschütz**, work on relation modules / the number of generators of `N^{ab}`.
- **S. Nakajima**, *Action of an automorphism of order `p` on cohomology groups of an
  algebraic curve* (J. Pure Appl. Algebra, 1985) — `H¹`/differentials as modular
  representation, wild case.
- **E. Kani**, **Valentini–Madan** — Galois module structure of differentials/`H¹` for
  covers in characteristic `p`.
- **Bleher–Chinburg–Kontogeorgis**, **Karanikolopoulos–Kontogeorgis** — Galois structure of
  holomorphic differentials of curves with a `p`-group action; explicit indecomposable
  summand descriptions in the cyclic and elementary-abelian cases.

The specific question "is `H¹(X, F₂)` indecomposable when `X/G = P¹` is branched at exactly
3 points and `G` is a 2-generated 2-group" does not appear to be directly answered in this
literature; the cyclic case (§4a) is the natural point of contact.

---

## 6. Computational experiments that would inform the proof

All cheap with the existing tooling (`torsion_module.sage` + MeatAxe):

1. **Cyclic-case module type.** For `G = C_{2^k}`, print the uniserial length of
   `H¹(X, F₂)` (should equal `dim = 2g`, confirming `F₂[G]`-cyclic). If always cyclic, (§4a)
   is essentially proved.
2. **Loewy / socle series** of `H¹(X, F₂)` across the census — look for a uniform pattern
   (e.g. socle dim = number of branch points with `σ_b ≠ 1`, or similar) that suggests the
   gluing mechanism.
3. **`Ext¹` obstruction.** For metacyclic `G`, compute the extension class of
   `0 → puncture → ? → H¹(X) → 0` (or the relevant class in (†)) and check it is "non-split
   in a single block."
4. **Stress test the hypotheses.** Does a decomposable `H¹` ever appear for (i) a
   non-2-generated 2-group cover (`≥ 3` branch points, so monodromy from `F₃`), or (ii)
   `X/G` of positive genus? Finding where indecomposability *first fails* would identify
   exactly which hypotheses the proof must use (2-generation? genus-0 base? exactly 3 branch
   points?).

Experiment 4 is the most valuable: it turns "why is it always indecomposable" into "what is
the minimal hypothesis," which is usually how such a proof is found.
